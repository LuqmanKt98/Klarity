
# Klarity - Minimal Mental Wellness App

Klarity is a calm, minimal mental wellness MVP designed with a mobile-first philosophy. It prioritizes emotional clarity through structured conversations, instant affirmations, and passive insight tracking.

## Core Features

- **Personalized Chat**: Reflective AI simulation that adapts based on selected healing modalities (Somatic, Mindfulness, CBT, etc.).
- **Instant Oasis**: A swipable, high-aesthetic feed of affirmations to ground you instantly.
- **Insights Dashboard**: Passive tracking of your "Mood Current" and themes extracted from your conversations.
- **Privacy First**: All data is stored locally in your browser's `localStorage`. No data leaves your device.
- **Premium Design**: Fluid animations, watery gradients, and low-density UI for maximum psychological safety.

## Tech Stack

- **React 18+** with **TypeScript**
- **Tailwind CSS** for ultra-clean styling
- **Lucide React** for minimal iconography
- **Recharts** for intuitive mood visualization
- **LocalStorage** for data persistence

## Getting Started

1. Ensure you have Node.js installed.
2. Run `npm install` (or your preferred manager).
3. Run `npm run dev` to start the development server.
4. Open the local link in your browser.
